CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Style: is an attribute to change the color
  • Incline styles apply to a specific HTML tag.
  • Internal style sheets contains CSS rules for the webpage in the head section of the HTML file.
  • ID's are use to identify specific elements.
  • External CSS style sheets are preferred in most cases because it allows developers to keep all their CSS rules in a seperate file which makes design changes easier.
